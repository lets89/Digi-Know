Clase 7: Arreglos (Arrays)

Objetivo

Almacenar múltiples valores en una sola variable y recorrerlos con bucles.

Temas

¿Qué es un arreglo?

Es una colección ordenada de elementos, como una lista.

Métodos básicos

  • push: agrega un elemento al final
  • pop: elimina el último elemento
  • length: devuelve la cantidad de elementos

Ejemplo en JavaScript

let frutas = ["manzana", "pera", "uva"];
console.log(frutas[1]); // pera

Práctica

Crea un arreglo con 5 nombres y muestra cada uno en consola:

let nombres = ["Ana", "Luis", "Marta", "Pedro", "Sofia"];
for(let i = 0; i < nombres.length; i++){
  console.log(nombres[i]);
}
