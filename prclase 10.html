Clase 10: Programación en HTML y CSS

Objetivo

Entender cómo se conecta la programación con una página web usando HTML, CSS y JavaScript.

Temas

Estructura básica de HTML

HTML define el contenido de la página.

Conexión con JavaScript

JavaScript permite agregar interactividad a los elementos HTML.

Ejemplo

<button onclick="saludar()">Haz clic</button>
<script>
  function saludar() {
    alert("¡Hola desde JavaScript!");
  }
</script>

Práctica

Crea una página que muestre un botón con un mensaje al hacer clic.
