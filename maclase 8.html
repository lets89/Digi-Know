Clase 8: Proporciones y porcentajes

Objetivo

Resolver problemas de proporciones y porcentajes.

Temas

Regla de tres

Permite relacionar proporciones para encontrar un valor desconocido.

Porcentaje y conversión decimal

Para convertir un porcentaje a decimal: % ÷ 100

  • 25% de 200 = 0.25 × 200 = 50

Práctica

  • Calcula el 15% de 80 = ______
  • Si 3 lápices cuestan $6, ¿cuánto cuestan 10? = ______
  • Un producto tiene 20% de descuento, precio original $50. ¿Precio final? = ______
