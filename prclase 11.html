Clase 11: Manejo básico del DOM

Objetivo

Modificar elementos HTML con JavaScript usando el DOM.

Temas

¿Qué es el DOM?

Es la representación estructurada de una página web que JavaScript puede modificar.

Métodos comunes

  • getElementById: selecciona un elemento por su ID
  • innerText: cambia el texto del elemento
  • value: obtiene el valor de un input

Ejemplo

<p id="mensaje"></p>
<button onclick="mostrarMensaje()">Mostrar</button>

<script>
  function mostrarMensaje() {
    document.getElementById("mensaje").innerText = "¡Hola Mundo!";
  }
</script>

Crea un formulario que muestre el nombre ingresado en pantalla.
