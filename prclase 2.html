Clase 2: Variables y tipos de datos

Objetivo

Aprender a declarar y usar variables.

Temas

¿Qué es una variable?

Es un espacio en memoria con un nombre que almacena un valor.

Tipos de datos

  • Número: 15, 3.14
  • Texto (string): "Luis"
  • Booleano: true / false

Ejemplo en JavaScript

let nombre = "Luis";
let edad = 15;
let estudiante = true;

console.log(nombre);
console.log(edad);
console.log(estudiante);

Práctica

Declara 3 variables con tus datos personales y muéstralas en consola.
