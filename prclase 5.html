Clase 5: Bucles (for y while)

Objetivo

Repetir instrucciones de forma controlada usando bucles.

Temas

Bucles for

Se usa cuando conocemos cuántas veces queremos repetir algo.

for (let i = 1; i <= 5; i++) {
  console.log("Iteración:", i);
}

Bucles while

Se usa cuando la condición determina cuándo detenerse.

let i = 1;
while (i <= 5) {
  console.log("Iteración:", i);
  i++;
}

Práctica

Crea un bucle que muestre los números del 1 al 10:

for (let i = 1; i <= 10; i++) {
  console.log(i);
}
