Clase 9: Expresiones algebraicas

Objetivo

Introducir el uso de letras para representar números y simplificar expresiones.

Temas

Variables y términos algebraicos

Las letras como x, y, a, b representan números desconocidos o cambiantes.

Suma y resta de monomios

Se combinan los términos semejantes (los que tienen la misma variable).

  • 3x + 2x = 5x
  • 4a - 2a = 2a

Práctica

  • Simplifica: 4a + 3a - 2a = ______
  • Simplifica: 5x - 3x + 2x = ______
  • Evalúa si x = 2: 3x + 4 = ______
