Clase 12: Proyecto final

Objetivo

Aplicar todos los conocimientos adquiridos en un proyecto completo.

Temas

Planificación y codificación

Define el propósito, estructura y funcionalidades del proyecto.

Diseño básico con HTML, CSS y JS

  • HTML: estructura del contenido
  • CSS: estilos visuales
  • JavaScript: interactividad y lógica

Ejemplo de proyecto

Registro de usuarios

  • Formulario con campos de nombre y edad
  • Estilos básicos para botones y campos
  • Validación y mensaje final con JavaScript

Práctica

Desarrolla tu propio mini proyecto. Algunas ideas:

  • Calculadora
  • Lista de tareas
  • Registro de usuarios
