Clase 6: Funciones

Objetivo

Organizar el código en bloques reutilizables mediante funciones.

Temas

Definición de funciones

Una función es un bloque de código que realiza una tarea específica.

Parámetros y retorno

Las funciones pueden recibir valores (parámetros) y devolver resultados (return).

Ejemplo en JavaScript

function saludar(nombre) {
  return "Hola, " + nombre + "!";
}
console.log(saludar("Luis"));

Práctica

Crea una función que calcule el área de un triángulo:

function areaTriangulo(base, altura) {
  return (base * altura) / 2;
}
console.log(areaTriangulo(5, 4));
