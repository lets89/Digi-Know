Clase 7: Mínimo común múltiplo y máximo común divisor

Objetivo

Calcular el MCM y el MCD de números usando descomposición en factores primos.

Temas

Descomposición en factores primos

Se usa para encontrar el MCM y el MCD de dos o más números.

Aplicaciones en fracciones

El MCM se usa para sumar o restar fracciones con distinto denominador. El MCD se usa para simplificar fracciones.

Ejemplos

  • MCM(6, 8) = 24
  • MCD(12, 18) = 6

Práctica

  • MCM(9, 12) = ______
  • MCD(8, 20) = ______
  • MCM(4, 10) = ______
  • MCD(15, 25) = ______
