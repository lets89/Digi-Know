Clase 8: Objetos

Objetivo

Entender cómo agrupar datos relacionados usando objetos en JavaScript.

Temas

¿Qué es un objeto?

Es una estructura que permite almacenar múltiples valores relacionados mediante pares clave:valor.

Sintaxis básica

let persona = {
  nombre: "Luis",
  edad: 16,
  carrera: "Informática"
};
console.log(persona.nombre);

Práctica

Crea un objeto llamado auto con las propiedades marca, modelo y color:

let auto = {
  marca: "Toyota",
  modelo: "Corolla",
  color: "Rojo"
};
console.log(auto.marca);
