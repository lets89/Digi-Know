Clase 4: Condicionales (if, else)

Objetivo

Tomar decisiones dentro del código usando estructuras condicionales.

Temas

Estructura condicional

Permite ejecutar diferentes bloques de código según una condición.

  • if: si se cumple la condición
  • else: si no se cumple
  • else if: para evaluar condiciones adicionales

Operadores de comparación

  • > mayor que
  • < menor que
  • == igual
  • != diferente

Ejemplo en JavaScript

let edad = 18;
if (edad >= 18) {
  console.log("Eres mayor de edad");
} else {
  console.log("Eres menor de edad");
}

Práctica

Escribe un programa que diga si un número es positivo, negativo o cero:

let numero = 5;
if (numero > 0) {
  console.log("Positivo");
} else if (numero < 0) {
  console.log("Negativo");
} else {
  console.log("Cero");
}
