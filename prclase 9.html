Clase 9: Entrada y salida de datos

Objetivo

Aprender a interactuar con el usuario mediante funciones de entrada y salida.

Temas

Funciones en JavaScript

  • prompt(): solicita información al usuario
  • alert(): muestra un mensaje emergente
  • console.log(): imprime en la consola

Ejemplo

let nombre = prompt("¿Cuál es tu nombre?");
alert("Bienvenido, " + nombre);

Práctica

Pide al usuario su edad y muestra un mensaje según si es mayor o menor de edad:

let edad = prompt("Ingresa tu edad:");
if (edad >= 18) {
  alert("Eres mayor de edad");
} else {
  alert("Eres menor de edad");
}
